The ETH4K is an FPGA development board based around Lattice Semiconductor's iCE40HX4K. It's a part of the Disco Board&trade; family of general-purpose development boards with key features including:
* 90 fully programmable GPIO
* 4 analog inputs (12-bit resolution)
* 10BASE-T/100BASE-TX Ethernet controller
* USB-UART bridge
* 1MiB of flash memory
* Storage of up to four separate bitstreams
* Multiple power options
* Industrial temperature range (-40°C to 85°C)
