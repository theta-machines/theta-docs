    \section{Features}
    \begin{itemize}
        \item 90 fully programmable GPIO
        \item 4 analog inputs (12-bit resolution)
        \item 10BASE-T/100BASE-TX Ethernet Controller
        \item USB-UART bridge
        \item 1MiB of flash memory
        \item Stores up to four separate bitstreams
        \item Multiple power options
        \item Industrial temperature range (-40°C to 85°C)
    \end{itemize}

    \begin{itemize}
        \item Prototyping industrial hardware
        \item Factory automation
        \item Motion control devices
        \item Embedded systems
        \item Ethernet-to-GPIO
        \item USB-to-GPIO
    \end{itemize}

    \section{Description} \label{section:description}
    The ETH4K is an open-source FPGA development board based on the iCE40HX4K. It is part of the Disco Board\texttrademark{} family and contains all of the necessary hardware to implement an RTMC card. The board includes several robust communication methods such as Ethernet, USB, and many GPIO. It can store up to four separate bitstreams, and is programmed over a single USB cable.
